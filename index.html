Projects

Profpocalypse: Purdue Edition

In this 2D turn-based battle game, it is the year 3025 at Purdue University… In an attempt to keep tuition frozen, the professors from 1000 years prior have been brought back to life, but with a twist - they’re now evil cyborgs (most of them).

This project was created with a group of 6 developers as part of a group project for Purdue's CS307 - Software Engineering I course. Built on Godot and utilizing GDScript, it contains various elements of video game development - a save file system, music and SFX integration, battle and enemy logic, NPC interaction, and more.

UNIX Shell

A custom UNIX shell with most capabilities of the standard UNIX shell - command flow was implemented through Lex/Yacc, and executed through a combination of C and C++. Has a full line reader as well as implementation for subshells, wildcards, etc.

Basic Messaging Application

This app is a standard messaging application utilizing Java Input/Output Stream to simulate server/client interaction.

Simple C Compiler

A Simple C Compiler created through use of Lex, Yacc, and x86-64 Assembly - able to interpret and compile C programs with limited variable types and operations.
